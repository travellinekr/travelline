## 작업 일시
2026-03-03

## 구현 내용

### 전체 흐름
```
/explore
├── [지역 탭] 일본 | 중국·대만 | 동남아 | 이색·장거리
├── [도시 그리드] 이미지카드 클릭 →
│   ├── [도시 헤더] 대표 이미지 + 도시명/설명
│   ├── [카테고리 탭] 맛집|숙소|교통|쇼핑|투어&스파|기타|공유플랜|여행가이드
│   └── [좌: 카드 리스트] + [우: 지도 + 상세 패널]
```

### 카드 리스트 (좌)
- **맛집**: `RESTAURANTS_DATA` 도시별 실 데이터 표시 (이모지+이름+가격대+미슐랭)
- **숙소**: `ACCOMMODATIONS_DATA` 도시별 실 데이터 표시 (타입+설명+태그)
- 카드 클릭 시 선택 하이라이트(오렌지 테두리+배경) + 우패널 오픈
- 동일 카드 재클릭 시 선택 해제

### 상세 패널 (우, md 이상에서만 표시)
- **상단**: Google Maps iframe (`maps.google.com/maps?q=lat,lng&output=embed`)
- **하단**: 카드 상세 정보
  - 맛집: 대표메뉴, 가격대, 영업시간, 미슐랭, 특징 태그
  - 숙소: 설명, 체크인/아웃, 태그

### 교통/쇼핑/투어&스파/기타/공유플랜/여행가이드
- 현재 빈 상태("아직 콘텐츠가 없어요") + 추후 데이터 추가 예정
